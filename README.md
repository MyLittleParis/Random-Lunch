# Random Lunch Bot

## Stack

- Node v16
- TypeScript

## Installation

2. Building the dev environment:

```sh
docker-compose up
```

> `-d` for detach mode

## Running the application

1. Connect to the container:

```sh
docker-compose exec app bash
```

2. Run the command:

> Note: you need to export the required environment variables for now before running the app

```sh
npm start
```
